# Azure Data Factory Assignment

## Objective
To understand Azure cloud concepts and build a complete data pipeline using Azure Storage Account and Azure Data Factory.

## Tasks Completed

### Task 1: Resource Group Creation
- Created Azure Resource Group.

### Task 2: Storage Account Setup
- Created Storage Account and Blob Container.

### Task 3: Azure Data Factory Configuration
- Created Linked Services.
- Created Source and Destination Datasets.
- Implemented Get Metadata Activity.

### Task 4: Pipeline Development
- Built a pipeline to move data from source to destination.

### Task 5: Pipeline Execution
- Successfully executed the pipeline using Debug mode.

### Task 6: IAM Roles
- Configured required access permissions.

## Mini Project

### Problem Statement
Build a complete pipeline that reads a CSV file from Blob Storage and processes it using Azure Data Factory.

### Solution
- Source: CSV file in Blob Storage
- Activities Used:
  - Copy Data
  - Get Metadata
- Destination: New Blob Storage Location

### Output
- Pipeline executed successfully.
- Data copied to destination.
- Metadata validated successfully.
