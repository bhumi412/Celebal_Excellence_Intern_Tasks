# Week 2 – SQL Database Analysis

## Overview

This project is part of my Data Engineering Internship and focuses on analyzing e-commerce business data using SQL. The assignment is based on the ShopEase database schema, which consists of customers, products, orders, and order items tables.

The objective was to write SQL queries to extract meaningful business insights, understand customer behavior, analyze product performance, and demonstrate knowledge of database concepts.

---

## Database Schema

The database contains the following tables:

* Customers
* Products
* Orders
* Order_Items

### Relationships

* One Customer can place multiple Orders.
* One Order can contain multiple Order Items.
* One Product can appear in multiple Order Items.
