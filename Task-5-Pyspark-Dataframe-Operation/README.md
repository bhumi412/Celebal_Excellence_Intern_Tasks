# Week-5: PySpark DataFrame Operations

## Objective
Perform data cleaning, transformation, filtering, and aggregation using PySpark DataFrames.

## Topics Covered
- Spark Fundamentals
- In-Memory Computing
- Data Cleaning
- Handling Null Values
- Removing Duplicates
- Filtering Data
- GroupBy & Aggregations
- Shuffle & Wide Transformations
- Timestamp Conversion

## Tasks Performed
- Removed duplicate records
- Handled null values using `.na.fill()`
- Filtered records using conditions
- Applied `groupBy()` and aggregation functions
- Converted timestamp data types
- Calculated min, max, average, and total revenue
- Built an end-to-end Spark processing pipeline
